# Silent Depth Command - Phase 2

Phase 2 adds the first playable navigation loop for the submarine command prototype.

## Included
- Full source project
- Static build ready for GitHub Pages in `build/`
- Visible version + build date/time in the interface
- PT-BR / EN / ES localization
- Playable tactical navigation: speed, heading, dive/surface, depth, and basic resource drain

## Run locally
Open `index.html` in a modern browser.
